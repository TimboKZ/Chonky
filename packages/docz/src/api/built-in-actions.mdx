# Built-in actions

Note that `EssentialActions`, `DefaultActions` and `ExtraActions` as combined into a
single object called `ChonkyActions`:

```ts
export const ChonkyActions = {
    ...EssentialActions,
    ...DefaultActions,
    ...ExtraActions,
};
```

This object is exported directly from `chonky`:

```ts
import { ChonkyActions } from 'chonky';

console.log(ChonkyActions.OpenFiles);
```

See [_Built-in actions_](/file-actions/built-in-actions) page for more info.
